# User

Get user data

**URL** : `/accounts/user/<int:pk>`

**Method** : `GET`

**Auth required** : YES 

**Permissions required** : NONE

## Success Responses

**Code** : `200 OK`


```json
{
  "id": 2,
  "email": "savytskyi.work@gmail.com",
  "name": "Anton",
  "surname": "Savytskyi"
}
```

## Error Responses

**Code** : `400`
